📘 ¿Qué es la inyección SQL?

Es una vulnerabilidad que permite a un atacante interferir con las consultas que una aplicación hace a su base de datos. Ocurre cuando se introducen datos maliciosos en una consulta SQL sin la debida sanitización.

Ejemplo clásico: En un login, si el campo usuario recibe ' OR '1'='1' -- , la consulta se convierte en:

SELECT * FROM usuarios WHERE username = '' OR '1'='1' -- ' AND password = '...'

Esto siempre devuelve resultados porque '1'='1' es verdadero y el resto se comenta. Así se puede acceder sin contraseña.
