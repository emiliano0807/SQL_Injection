Evaluación de Inyección NoSQL

Manipulación de estructuras JSON en bases de datos documentales (MongoDB).

Riesgo de Configuración: La API acepta objetos JSON de forma nativa sin validar estrictamente el tipo de dato, exponiendo operadores.
Consulta MongoDB Generada

Plantillas de Prueba (Payloads)

Evasión mediante $ne (Not Equal)
User: {"$ne": ""}
Pass: {"$ne": ""}

Busca el primer documento donde las credenciales no estén vacías.

Evasión mediante $gt (Greater Than)
User: {"$gt": ""}
Pass: {"$gt": ""}

Filtra documentos cuyos valores sean mayores a una cadena vacía.

Evaluación de Inyección LDAP

Manipulación de filtros LDAP para acceder a directorios.

Vulnerabilidad detectada: El filtro LDAP se construye concatenando la entrada directamente: (&(uid={usuario})(userPassword={contraseña})).
Filtro LDAP Generado

Plantillas de Prueba (Payloads)

Wildcard básico
*

El asterisco coincide con cualquier valor en LDAP.

Bypass con OR lógico
admin)(|(uid=*

Cierra el filtro original e inyecta un OR general.

Auditoría GraphQL

Análisis de esquemas y permisos en APIs basadas en GraphQL.

Vulnerabilidad detectada: El endpoint tiene introspección habilitada, permitiendo enumerar todo el esquema.
Respuesta JSON del Servidor

Plantillas de Introspección

Extracción de Entidad Completa

Mapeo Estructural (__schema)
